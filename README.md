# Diabetes_Prediction

## Introduction:
Diabetes is a serious health condition that requires early detection to prevent complications. This project uses machine learning techniques to predict diabetes based on patient health data such as glucose level, BMI, and age. By comparing multiple ML models, the study aims to identify the most accurate model for reliable diabetes prediction and improved clinical decision support.

## Objective:
1. To analyze a diabetes dataset and understand key risk factors
2. To build and compare multiple machine learning models
3. To evaluate model performance using standard metrics
4. To identify the most important features contributing to diabetes prediction

## Model framework:
- Data Collection: Patient medical records
- Pre-Processing: Data cleaning and normalization
- EDA: Statistical analysis and visualization
- Feature Selection: Identification of key risk factors
- Model Training: Logistic Regression, KNN, SVM, Decision Tree, Random Forest
- Evaluation: Accuracy, Precision, Recall, F1-Score
- Prediction: Diabetic / Non-Diabetic classification
- Results: Model comparison and insights

## Result:
- Multiple machine learning models were trained and evaluated for diabetes prediction using clinical features such as glucose level, BMI, age, insulin, and blood pressure.
- Data preprocessing, feature scaling, and train–test splitting improved model stability and performance.
- Among all evaluated models, ensemble and tree-based classifiers outperformed traditional algorithms.
- The best-performing model achieved high classification accuracy with balanced precision and recall, indicating reliable diabetes detection.
- Confusion matrix analysis showed strong true positive and true negative rates, minimizing misclassification.
- Feature importance analysis revealed that Glucose, BMI, Age, and Insulin were the most influential predictors of diabetes.
